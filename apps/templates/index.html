API Challenge Info

FEBRUARY 27 - MARCH 24

The Vainglory API powered by MadGlory is now live! If you’ve got some development chops, you can now create your very own app that uses live Vainglory data to win the respect of your fellow players and even some prizes!

Register by yourself or with a team of up to 4 for the API Challenge by creating a piece of software that uses Vainglory data. The contest starts on February 27th, and will run until March 24th at 11:59 PM EST. A grand prize winner for best overall app will be crowned, along with the top two most impressive entries for each category. You can enter as many apps as you’d like; you just need to fill a separate submission for each one.
